## 概述

铝合金智能设计系统（Alalloy Agent）v0.1.0 正式发布。本系统以多专家 AI 协作为核心，将材料数据库检索、机器学习性能预测、CALPHAD 热力学计算、设计报告生成等环节整合为一套端到端的对话式工作流，旨在显著缩短铝合金成分设计的迭代周期。

---

## 核心功能模块

### 1. 智能成分推荐（数据专家）

- 接入 IDME 历史材料数据库，检索同类 Al-Si-Mg 体系合金的成分与性能记录
- 当数据库无满足目标的样本时，提供量化差距分析（指出具体性能缺口）和成分瓶颈识别，推荐 2 组突破性候选成分
- 输出候选成分推荐表格，包含设计依据与数据库对比说明

### 2. 机器学习性能预测（分析专家 · ONNX）

- 集成压铸铝合金 ONNX 预测模型，对多组候选成分并行推理
- 输出抗拉强度（UTS）、延伸率（EL）等关键力学性能预测值
- 支持多成分批量对比，一次响应给出所有结果

### 3. CALPHAD 热力学计算（分析专家 · CalphaMesh）

支持 **7 类热力学任务**，分两阶段执行：

| 类型     | 任务                            | 说明                         |
| -------- | ------------------------------- | ---------------------------- |
| 核心三项 | 单点相计算（point）             | 指定温度下的相组成与摩尔分数 |
|          | Scheil 凝固模拟（scheil）       | 凝固路径、凝固区间、热裂倾向 |
|          | 温度线扫描（line）              | 温度区间内相组成变化         |
| 扩展四项 | 热力学性质（thermo_properties） | G、H、S、Cₚ 随温度曲线      |
|          | 二元相图（binary）              | 两种元素的完整相图           |
|          | 三元等温截面（ternary）         | 指定温度下的三元相图         |
|          | 熔点/沸点计算（boiling_point）  | 熔沸点预测                   |

- 耗时较长的任务（Scheil / 二元相图 / 三元截面）配有人工确认机制（HITL），防止误操作
- 快速任务（单点 / 线扫描 / 熔点等）直接执行，无需确认

### 4. 设计报告生成（报告专家）

- 汇总数据库分析、性能预测、热力学计算的全部结果，自动生成结构化 Markdown 报告
- 报告包含：任务背景、成分对比、热力学特性、工艺建议、风险提示与推荐结论
- 支持一键导出 PDF，样式参照公司文档规范（含页眉、页脚、LOGO）

## 交互方式

- **对话式工作流**：通过自然语言描述设计需求（如"设计 UTS ≥ 330 MPa、延伸率 ≥ 15% 的压铸铝合金"），系统自动拆解任务并逐步推进
- **引导挂件**：每个阶段完成后提供交互选项（性能预测 / 热力学分析 / 生成报告 / 调整成分），用户点击即可流转，无需手动输入指令
- **结果面板**：计算结果以专业图表（Scheil 曲线、相图、热力学性质曲线等）直观呈现，支持数据下载
- **会话恢复**：刷新或重连后可完整恢复历史对话状态与计算结果
